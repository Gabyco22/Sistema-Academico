# Sistema-Academico
Un sistema de gestión académica desarrollado en **Python + SQLite**, que permite:
- Registrar alumnos y cursos
- Inscribir alumnos en cursos
- Gestionar notas
- Generar reportes (promedios, mejores alumnos, etc.)
- Visualizar datos con gráficos en consola y matplotlib

## Características
- Menú interactivo en consola con colores y ASCII art
- Base de datos SQLite para persistencia
- Reportes automáticos de notas y cursos
- Visualización inicial con banner y estadísticas
